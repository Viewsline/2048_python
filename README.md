# 2048 Pygame

## Consignes pour jouer:
- **Lancer le jeu** : exécutez le fichier `main.py`.
- **Pour jouer** : utilisez les flèches directionnelles du clavier.
- **Pour défaire un coup** : utilisez la touche `Retour arrière` (Backspace).
- **Pour refaire un coup** : utilisez la touche `Entrée`.

## Version Web (Pygbag)
Ce jeu a été adapté et optimisé pour le Web grâce à [Pygbag](https://pygame-web.github.io).  
La boucle asynchrone permet d'exécuter le jeu directement dans le navigateur.

Dès que vous effectuez un `git push` sur la branche `main` ou `master`, un workflow **GitHub Actions** compile automatiquement le projet et le déploie sur vos **GitHub Pages**. Il sera alors jouable par n'importe qui depuis l'URL de vos pages statiques.
